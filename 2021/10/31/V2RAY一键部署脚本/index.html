V2RAY一键部署脚本

此安装脚本除了可以安装V2以外，亦可以安装SS，MTP，Socks5，简单好用； 一键安装命令： Debian/Ubuntu 
apt-get update
apt-get install curl -y
bash <(curl -s -L https://git.io/v2ray.sh)
Centos 
yum update
yum install curl -y
bash <(curl -s -L https://git.io/v2ray.sh)
根据提示选择需要的选项安装即可； 管理面板命令：
